# Rep2 selected cells

A csv file containing the coordinates and cluster label for each cell of
the selected subset of rep2.

## Usage

``` r
data(rep2_clusters)
```

## Format

A data frame with 1815 rows and and 6 variables:

- anno:

  the provided cell type annotation

- cluster:

  cluster label

- x:

  x coordinates

- y:

  y coordiantes

- cells:

  cell id

- sample:

  sample id
