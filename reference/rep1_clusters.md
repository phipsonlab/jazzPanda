# Rep1 selected cells

A data frame file containing the coordinates and cluster label for each
cell of the selected subset of rep1.

## Usage

``` r
data(rep1_clusters)
```

## Format

A data frame with 1705 rows and 6 variables:

- anno:

  the provided cell type annotation

- cluster:

  cluster label

- x:

  x coordinates

- y:

  y coordiantes

- cells:

  cell id

- sample:

  sample id
